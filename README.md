# InstaSolve Landing Page

Landing Page for my App [InstaSolve](https://github.com/HarshNarayanJha/Insta-Solve) made in Vue.js and Vite!

## Project Setup

```sh
bun install
```

### Compile and Hot-Reload for Development

```sh
bun dev
```

### Compile and Minify for Production

```sh
bun build
```
